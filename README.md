# Tennis-Rückhand · 25-Min-Sportstunde Klasse 10

Interaktive Begleit-Website zu einer 25-minütigen Sportstunde **Einführung beidhändige Tennis-Rückhand** für eine 10. Klasse Gymnasium in Nordrhein-Westfalen.

**Live:** https://fabianxneumann.github.io/tennis-rueckhand-stunde/

## Inhalt

- Live-Stundentimer mit automatischen Phasenwechseln (25 Min, 9 Phasen)
- 5 Stationen mit Niveau-Differenzierung (◐ leicht / ◐◐ mittel / ◐◐◐ anspruchsvoll)
- Trainer:innen-Briefing mit External-Focus-Cue-Cards (Wulf 2013)
- Theorie-Abschnitt mit 16 wissenschaftlichen Quellen (APA 7)
- Material-Liste, Sicherheits-Checkliste (DGUV / BASS NRW), Hallenplan-SVG
- Trainer-Beobachtungsbogen + SuS-Reflexionskarten
- Druckansichten für A5-Stationenkarten und A4-Bögen
- Dark-Mode-Toggle, mobile-responsive

## Tech

Pure HTML5 + CSS3 + Vanilla JavaScript — keine externen Dependencies, keine Frameworks. Lokal öffenbar via `file://` oder `python3 -m http.server`.
